# Chatroom Template

終端機風格直播聊天室模板集合。

專為直播使用設計，支援 Streamlabs 與 OneComme，風格靈感來自終端機、系統控制台與科幻通訊介面。

---

## 功能特色

- 終端機風格介面
- 支援 Streamlabs
- 支援 OneComme 9.0
- 直式聊天室版型
- 橫式 Wide Overlay 版型
- 無外部依賴
- 輕量化設計
- 可自由修改與擴充

---

## 內含模板

### Chatroom Sync V1.0

標準終端機風格聊天室。

支援平台：

- Streamlabs

---

### Chatroom Sync wide-overlay V1.0

適合遊戲直播使用的橫式版型。

推薦用於：

- FPS 遊戲
- Helldivers 2
- Euro Truck Simulator 2
- 一般遊戲直播

支援平台：

- Streamlabs

---

### onecomme Chatroom Sync V1.0

OneComme 專用版本。

支援平台：

- OneComme 9.0

---

## 使用方式

### Streamlabs

1. 下載需要使用的模板資料夾
2. 將 HTML、CSS、JS 匯入 Streamlabs
3. 設定 Browser Source 或 Custom Widget
4. 完成

### OneComme

1. 下載 OneComme 版本模板
2. 匯入 OneComme 9.0
3. 完成設定
4. 開始使用
